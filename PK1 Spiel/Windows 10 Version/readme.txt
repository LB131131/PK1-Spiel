1. SYSTEM
    -Betriebssystem: Windows 10 Professional
    -Gerät: PC mit 27 Zoll Bildschirm
    -Python Version: 3.11

2. VERWENDUNG UND START:
    -Stellen sie sicher, dass die Dateien: teilfunktionen.py, cheats.py, farben.py, gegner.py, map.py, savegame.py,scoreboard.py, spiel.py, spieler.py, items.py und konstanten.py im selben Ordner liegen
    -starten sie in der Windows Eingabeaufforderung die datei "spiel.py"

3. STEUERUNG:
    -Bewegung: w,a,s,d (gedrückthalten möglich)
    -Speichern: k
    -Laden: l
    -Custom-Menü: c
    -Beenden: b

4. FARBEN:
    -buchstabe, weiß: nicht freigestaltetets Ziel
    -grün: freigeschaltetes Ziel
    -rot: Gegner
    -blau: Item oder Waffe

5. BUCHSTABEN:
    -☺ : Spieler
    -p : Tor zum nächsten Level
    -Z : Zombie
    -Ge: Geist
    -Sk: Skelett
    -Go: Golem
    -H : Herz
    -B : Batterie
    -D : Dolch
    -L : Laserschwert
    -Sc: Schwert
    -F : Faust
